# 【基本フォント】Noto Sans CJK / 源ノ角ゴシック (源真ゴシック)

源真ゴシックは、源ノ角ゴシックベースのフォントです。

- http://jikasei.me/font/genshin/
- `href="GenJpFont-X.css"`
- font-family:
  - 欧文部分
    - "GenShinGothic-A-w"
  - 欧文以外(日本語+記号)
    - "GenShinGothic-w"
    - 欧文部分はMgen+に挿げ替えが可能です。
- 形式 Extensions
  - ttf,eot,woff2,woff
